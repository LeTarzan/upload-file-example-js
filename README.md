# Upload files to AWS S3


Upload file app developed with NodeJS and ReactJS

# Initial configurations

### Frontend

Access directory **Frontend** and run:

```console
foo@bar:~$ npm install
foo@bar:~$ npm start
```

### Backend

In **Backend** directory create **.env** based on **.env_example** after run:

```console
foo@bar:~$ npm install
foo@bar:~$ npm run dev
```

Features:

* List all save files;
* Add files;
* Delete files;
* File link;
